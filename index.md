## üìñ Overview

Requestal is a Chrome DevTools side-panel extension designed for users who want to perform human-driven security testing. With Requestal, you can easily capture, modify, validate, and replay HTTP requests directly from your browser. The extension offers semantic diffing and RFC-aware formatting, simplifying the process of analyzing web traffic and testing applications.

## üõ†Ô∏è Features

- **Capture HTTP Requests:** Easily intercept and view HTTP requests made by your web applications.
- **Modify Requests:** Change headers, parameters, and request bodies to test different scenarios.
- **Replay Requests:** Send modified requests back to the server to see how the application responds.
- **Semantic Diffing:** Compare requests visually to understand changes and impacts clearly.
- **RFC-Aware Formatting:** Get formatted requests and responses compliant with HTTP standards.
- **User-Friendly Interface:** Designed with ease of use in mind, perfect for users without technical backgrounds.

1. Open your Chrome browser.
2. Go to the Extensions page by typing `chrome://extensions/` in the address bar.
3. Enable "Developer mode" by flipping the switch in the top right corner.
4. Click on the "Load unpacked" button.
5. In the popup, select the folder where you downloaded the extension.
6. Requestal will now appear among your extensions and is ready for use.

## üîé Using Requestal

1. Open Chrome and navigate to a website where you want to test HTTP requests.
2. Open the Chrome DevTools by right-clicking on the page and selecting "Inspect" or pressing `Ctrl+Shift+I`.
3. Click on the Requestal panel on the right side of the DevTools.
4. Begin capturing requests as you interact with the website.
5. Modify any request as needed using the simple interface.
6. Replay requests to see changes in response or behavior.
